Connection Guide

Pair Device to Account

Link your physical WashTrac CodeBox keypad to your online account using a secure 6-digit pairing code generated from the dashboard.

1

Log In to the Dashboard

Open the WashTrac client portal and sign in to your account.

  • Go to thewashtrac.com
  • Select Client Login
  • Enter your email and password
Screenshot Placeholder
Client Login Screen
2

Add or Select a Device

Create a new device or open an existing device you want to pair.

  • Open the Add Device menu
  • Enter a device name
  • Save the device
Screenshot Placeholder
Add Device Screen
3

Generate the 6-Digit Pairing Code

WashTrac will display a temporary pairing code.

  • Locate the 6-digit code
  • Write it down or keep the page open
Screenshot Placeholder
Example Pairing Code: 123456
Pairing codes are temporary and used only to securely link one physical device.
4

Enter Programming Mode on the Keypad

At the CodeBox keypad, hold the BACK button for several seconds.

  • Press and hold BACK
  • Programming menu appears
  • Select 3 = PAIR
Screenshot Placeholder
Programming Menu Showing 3 = PAIR
5

Enter the Pairing Code

Type the 6-digit code from the dashboard and press ENTER.

  • Enter all six digits
  • Press ENTER
  • Wait for confirmation
Screenshot Placeholder
Enter 6-Digit Pairing Code
6

Device Syncs Automatically

Once paired, the keypad downloads settings and active access codes.

  • Codes sync to the device
  • Activity begins logging to the dashboard
  • Reports and monitoring are enabled
Screenshot Placeholder
Dashboard Showing Connected Device

Pairing Complete

Your WashTrac device is now securely linked to your account and ready to track wash activity in real time.
