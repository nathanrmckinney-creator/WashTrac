Account Management Guide

Manage Sites & Users

Create and manage customer sites, invite users, and control who has access to each WashTrac location from the online dashboard.

1

Log In to the Dashboard

Open the WashTrac client portal and sign in to your account.

  • Go to thewashtrac.com
  • Select Client Login
  • Enter your email and password
2

Open Manage Sites

From the dashboard, open the site management area.

  • Open the dashboard menu or left-side navigation
  • Select Manage Sites or Sites
  • Review the sites already connected to your account

3

Create or Edit a Site

Use the Sites page to add a new wash location or update an existing one.

  • Click Add Site to create a new location
  • Enter the site name and site details
  • Save the site so devices, codes, and activity can be assigned correctly
Screenshot Placeholder
Add / Edit Site Form
Site information is created during registration for the first location. Use Manage Sites when adding more locations or updating existing site details.
4

Invite a User

Add users who need access to a specific WashTrac site.

  • Open the selected site
  • Choose the user or invite section
  • Enter the user’s email address
  • Select the correct role or permission level
Screenshot Placeholder
Invite User Screen
5

Assign User Permissions

Control what each user can view or manage.

  • Managers may need access to reports, activity, and site controls
  • Service or detail users may only need access to wash activity or assigned codes
  • Remove access when a user no longer needs the site
Screenshot Placeholder
User Permissions / Role Selection
6

Confirm Site Access

After saving, confirm that the user appears under the correct site.

  • Verify the user email is listed
  • Confirm the assigned role is correct
  • Make sure the user can only access the intended site or sites
