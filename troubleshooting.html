Support Guide

Troubleshooting

Use this guide to resolve common issues involving connectivity, pairing, access codes, display problems, and system status.

Device Shows Offline

The device may be powered but not currently connected to the internet.

  • Verify the unit has power
  • Check Wi-Fi settings in the dashboard
  • Confirm local internet service is working
  • Restart the device if needed

Pairing Code Does Not Work

Pairing codes are temporary and must be entered exactly.

  • Generate a new 6-digit code
  • Confirm you selected 3 = PAIR on the keypad
  • Re-enter the code carefully

Valid Code Does Not Start the Wash

The access code may be disabled or the output circuit may not be available.

  • Verify the code is active
  • Check remote shutdown settings
  • Check scheduled lockouts
  • Confirm E-STOP status

Display Is Blank

A blank display usually indicates a power or hardware connection issue.

  • Verify incoming power
  • Check display cable connections
  • Restart the unit

Cannot Connect to Wi-Fi

Incorrect passwords or weak signal strength are the most common causes.

  • Confirm the password is correct
  • Run a new wireless scan
  • Move closer to the wireless access point if possible

No Activity Appears in the Dashboard

If the device was offline, activity may upload after connectivity is restored.

  • Confirm the device is online
  • Refresh the dashboard
  • Verify the device is paired to the correct account
WashTrac is designed to continue operating locally even if internet connectivity is temporarily unavailable.

E-STOP Is Active

The system may intentionally prevent washes when an emergency stop circuit is detected.

  • Inspect the emergency stop circuit
  • Reset the wash equipment if required
  • Confirm status returns to normal
The WashTrac E-STOP input is intended for monitoring and status visibility and does not replace required safety systems.

Still Need Help?

If the issue continues after following these steps, contact WashTrac support and include screenshots and a description of the problem.
