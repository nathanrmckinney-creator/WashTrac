Features

Live Bay Status

Monitor the status of every wash bay in real time and know exactly when washes are active or idle.

Access Code Tracking

Track every access code usage including who used it, when it was used, and which bay was activated.

Mobile Visibility

View wash activity and system status from anywhere using your phone, tablet, or computer.

CodeBox Terminal

The WashTrac CodeBox is a full-screen mounted terminal designed for customer-facing access control at the bay. It provides a clean interface for code entry, wash activation, and platform interaction.

How It Works

Install a WashTrac CodeBox terminal on a post at each bay entrance. Users interact with the screen to enter their assigned access code and activate the wash. WashTrac records each activation and gives operators a clear view of activity, usage, and system status.

Operator Visibility

WashTrac gives operators a clean dashboard for reviewing wash activity, monitoring bay status, and managing access codes from anywhere.
