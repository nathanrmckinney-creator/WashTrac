1

Log In to the Dashboard

Open the WashTrac client portal and sign in to your account.

  • Go to thewashtrac.com
  • Select Client Login
  • Enter your email and password
2

Open the Wi-Fi Setup Card

From the dashboard, locate the Wi-Fi Setup card for the selected site and device.

  • Make sure the correct site is selected
  • Confirm the CodeBox is paired to that site
  • Use the Wi-Fi Setup card to manage connection settings
3

Search for Wireless Networks

Click the wireless search button to scan for available Wi-Fi networks near the device.

  • Select Search Wireless Networks
  • Wait for the available networks to load
  • Refresh the network status if the network does not appear
The device must be powered on and close enough to the wireless network for the scan to find it.
4

Select the Wi-Fi Network

Choose the correct network name from the list of available wireless connections.

  • Find the correct Wi-Fi name
  • Select that network
  • Confirm you selected the correct location network
5

Enter the Wi-Fi Password

Type the wireless password exactly as provided by the location or network administrator.

  • Enter the Wi-Fi password
  • Check capitalization and special characters
  • Select Connect Wi-Fi
Wi-Fi passwords are case-sensitive. An incorrect password will prevent the device from connecting.
6

Confirm Connection

After connecting, confirm that the dashboard shows the device as online.

  • Wait for the connection status to update
  • Confirm the device shows online
  • Verify activity can sync to the dashboard
7

If the Device Does Not Connect

If the device fails to connect, review the network information and try again.

  • Confirm the password is correct
  • Make sure the Wi-Fi network is available at the device location
  • Restart the device if needed
  • Run the wireless search again

Wi-Fi Setup Complete

Once connected, the device can sync access codes, activity logs, monitoring updates, and reporting data through the dashboard.
