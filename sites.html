Sites LOCATIONS

Add wash locations, pair physical keypads to the correct site, and manage user access for each location.

Add Site

Add Site creates the wash location. Enter the keypad pairing code or device ID here to link the physical WashTrac Keypad to this new site during setup. You can also pair or reassign keypads later under Site Management.

How User Access Works

Owner / Admin
Can add locations and control which users have access to each site.
Control
Email Invite
Invite employees or managers to access existing sites. Invited users only see the locations assigned to them.
Users
Pair Keypad
Use the keypad pairing code, serial number, or device ID to connect the physical WashTrac Keypad to the correct site.
Device
Sites are physical wash locations. Users are people invited by email. Keypads are physical devices and must be paired to the right site before they can report monitor status, validate wash codes, or log activity for that location.
